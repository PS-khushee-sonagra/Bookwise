# 📘 Reflection

## What I Learned

Through this course, I gained a strong understanding of the core concepts behind modern AI applications.

### Key Concepts
- **LLMs:** Large Language Models generate text by predicting the next token based on context.
- **Prompt Engineering:** Well-structured prompts improve response quality.
- **MCP (Model Context Protocol):** A standard way for AI models to interact with external tools and APIs.
- **Embeddings & Vector Databases:** Text is converted into vectors for semantic search, with FAISS used for efficient retrieval.
- **RAG (Retrieval-Augmented Generation):** Combines document retrieval with LLMs to produce accurate, context-aware responses.
- **RAG vs Fine-Tuning vs Prompting:** Learned when each approach is appropriate based on the use case.

## Applying the Concepts

I applied these concepts while building **BookWiseAI**, where I implemented:

- PDF text extraction
- Text chunking
- SentenceTransformer embeddings
- FAISS vector database
- Semantic search
- Gemini-powered answer generation
- Streamlit user interface

## Reflection

This course helped me understand both the theory and practical implementation of AI systems. Building BookWiseAI reinforced how LLMs, embeddings, vector databases, and RAG work together to create intelligent, real-world applications.
